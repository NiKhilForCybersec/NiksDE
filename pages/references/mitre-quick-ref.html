MITRE ATT&CK Quick Reference

Essential tactics, high-priority techniques, and detection data sources for rapid reference during detection development and interviews.

Tactics Overview

ID Tactic Goal Key Data Sources
TA0043 Reconnaissance Gather information Network traffic, DNS logs
TA0042 Resource Development Establish resources Domain registration, certificates
TA0001 Initial Access Get into network Email, web proxy, firewall
TA0002 Execution Run malicious code Process, command line, script
TA0003 Persistence Maintain foothold Registry, scheduled tasks, services
TA0004 Privilege Escalation Get higher permissions Process, authentication, tokens
TA0005 Defense Evasion Avoid detection Process, file, registry
TA0006 Credential Access Steal credentials Authentication, process, file
TA0007 Discovery Explore environment Process, command line, network
TA0008 Lateral Movement Move through network Authentication, network, SMB
TA0009 Collection Gather target data File, clipboard, email
TA0011 Command and Control Communicate with systems Network traffic, DNS, proxy
TA0010 Exfiltration Steal data Network traffic, cloud storage
TA0040 Impact Disrupt/destroy File, process, service

Quick Detection Patterns

Common Detection Patterns 
// T1059.001 - Encoded PowerShell
ProcessCommandLine has_any ("-enc", "-EncodedCommand", "-e ")
  AND ProcessCommandLine matches regex "[A-Za-z0-9+/=]{40,}"

// T1003.001 - LSASS Access
TargetProcessName == "lsass.exe" 
  AND SourceProcessName !in ("csrss.exe", "services.exe", "svchost.exe")

// T1053.005 - Scheduled Task Creation
EventID == 4698 
  OR ProcessCommandLine has_any ("schtasks", "/create")

// T1547.001 - Registry Run Key
RegistryKey has_any ("CurrentVersion\\Run", "CurrentVersion\\RunOnce")
  AND ActionType == "RegistryValueSet"

// T1110 - Brute Force
EventID == 4625 
  | summarize FailCount = count() by TargetAccount, IpAddress, bin(TimeGenerated, 5m)
  | where FailCount > 10

// T1558.003 - Kerberoasting
EventID == 4769 
  AND TicketEncryptionType == "0x17"
  AND ServiceName !endswith "$"

// T1021.001 - RDP Lateral Movement
EventID == 4624 
  AND LogonType == 10
  AND SourceNetworkAddress != "-"

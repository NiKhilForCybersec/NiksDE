NIST 800-53 Security Controls

Detailed security and privacy controls relevant to detection engineering, with focus on Audit and Accountability (AU), Incident Response (IR), and System and Information Integrity (SI) families.

NIST 800-53 Overview

NIST SP 800-53 Rev. 5 provides a catalog of security and privacy controls for federal information systems. It contains over 1,000 controls organized into 20 families. Detection engineers primarily work with AU, IR, SI, and CA control families.

Evidence: AU Control Implementation

Sentinel KQL - AU-6 Compliance Evidence 
// AU-6: Demonstrate audit review and analysis
// Generate evidence of automated audit analysis

// 1. Show active analytics rules (automated analysis)
let AnalyticsRuleCount = SecurityAlert
| where TimeGenerated > ago(30d)
| summarize UniqueRules = dcount(AlertName);

// 2. Show audit record review (alerts investigated)
let ReviewedAlerts = SecurityIncident
| where TimeGenerated > ago(30d)
| where Status in ("Closed", "Resolved")
| summarize ReviewedCount = count();

// 3. Show correlation capability (multi-source incidents)
let CorrelatedIncidents = SecurityIncident
| where TimeGenerated > ago(30d)
| mv-expand AlertIds
| summarize AlertCount = count() by IncidentNumber
| where AlertCount > 1
| count;

// Compliance summary
print 
    Control = "AU-6",
    ControlName = "Audit Review, Analysis, and Reporting",
    ActiveDetectionRules = toscalar(AnalyticsRuleCount),
    AlertsReviewed30Days = toscalar(ReviewedAlerts),
    CorrelatedIncidents = toscalar(CorrelatedIncidents),
    Status = "Implemented"
